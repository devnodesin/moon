### Documentation

View HTML documentation in your browser: [http://localhost:6006/doc/](http://localhost:6006/doc/)

Get Markdown documentation (for humans and AI coding agents) via curl:

```bash
curl "http://localhost:6006/doc/llms-full.txt"
```

Or view in browser: [http://localhost:6006/doc/llms-full.txt](http://localhost:6006/doc/llms-full.txt)

Refresh documentation cache:

```bash
curl -X POST "http://localhost:6006/doc:refresh" \
  -H "Authorization: Bearer $ACCESS_TOKEN" | jq .
```

**Response (200 OK):**

```json
{
  "message": "Documentation cache refreshed successfully"
}
```
